Stock Picking Cancel

Stock Picking Cancel helps to reverse the done picking,
allow to cancel picking and set it to draft stage.

Geo Technosoft

Features

Provides option to cancel Delivery order or Receipt and reset it to draft state.

Provides option to cancel stock move.

Configuration

Assign "Cancel Picking/Stock Move" group to the user.

Screenshots

Cancel button is visible in done state of picking to allow cancelling done picking.

Reset button is visible in cancelled state of picking to allow move into draft state.

Here we can see picking is now in draft state.

For cancelling moves in done state, Cancel action is given.

Selected moves are cancelled.
